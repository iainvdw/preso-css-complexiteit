    BEM

    Block, Element, Modifier

    
.block { ... }
.block__element { ... }
.block__element--modifier { ... }

    Wat is BEM?

    • Scheidt je pagina in componenten
    • De componenten bestaan uit Blocks en Elements
    • Blocks zijn containers
    • Elements zijn children
    • Directe relatie tussen Block en Element in class name

    Waarom BEM?

    • Forceert je tot nadenken over unieke componenten en relaties tussen elementen
    • Uniforme naamgeving over hele codebase
    • Relaties tussen elementen afleiden uit HTML
    • Niet tot nauwelijks nesten selectors
    • Uitermate geschikt voor doorontwikkeling op lange termijn

    Wat is ITCSS?

    • Dicteert source order
    • Daarmee dicteert het ook volgorde van specificiteit
    • Voorkomt verspringen door je stylesheet door sepcificiteit
